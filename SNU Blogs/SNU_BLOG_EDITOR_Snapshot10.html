Why can't I submit my blog?

At the moment, the SNU servers aren't live. You are using a beta build that also doesn't support publishing blogs. Try using a more modern build

What is greentext?

The greentext feature for the blog editor came from 4chan's greentexting system

Example:

> This arrow means greentext

< This arrows means redtext

> This is also known as a greentext story

What is bolding and how do I use it?

Bold text is used when you want a piece of content to stand out. It gives the text a strong tone. It shouldn't be used for the entire document, or else it loses its power

To use bolding, select the bold option in the formatting and font style menu.

What is subscript and how do I use it?

Supscript is formatted text that goes below the line. There is currently no definition listed here on how to use it

What is superscript and how do I use it?

Superscript is formatted text that goes above the line next to or behind a charcter. There is currently no definition listed here on how to use it

Example of superscript:
