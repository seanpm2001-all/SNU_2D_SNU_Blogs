Welcome to the official blog page for SNU!

Getting started

SNU blogs is the blog portion of SNU, where you get to read peoples blog posts. Don't get this confused with SNU books (page not yet available) SNU books have a different way of use.

With SNU blogs, anyone can write a story, and post it.

Recommended blogs

Since SNU is still in the beta phase, we only have 1 blog, and it is run by SNU: Templateaccountblogs

Originally, templateaccount was used for testing user profiles, and what a user profile is like, but we are in a changing dillema on profiles at the moment, and templateaccountblogs represents the basic HTML format.

It offers compatibility and default pages and examples for all of SNUs features. It is a developer testing account created by the creator of SNU, Sean Patrick Myrick

Templateaccountblogs is one of SNUs official blog pages, mainly consisting of developer logs that tell you about the day-to-day progress with SNU development, and what is going on with the developers.
